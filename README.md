# ArcadeEngine
> HTML5 Based Arcade Game Engine

## Feature
 * Providing Typescript Based Scripting / 타입스크립트 기반 스크립팅 지원
 * Support object-oriented programming / 객체 지향 개발 지원
 * support fps / fps 지원
 * Many components :
   * ArcadeObject
      * ArcadeSprite
   * ArcadeScene
   * and more components will be created
